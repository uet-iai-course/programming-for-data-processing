# Lập trình Xử lý Dữ liệu — Trường Đại học Công nghệ, ĐHQGHN

Tài liệu bài giảng và bài tập về nhà cho môn **Lập trình Xử lý Dữ liệu** tại Viện Trí tuệ Nhân tạo (IAI), Trường Đại học Công nghệ (UET), ĐHQGHN.

Slide bài giảng được xây dựng bằng [Reveal.js](https://revealjs.com/). Notebook minh hoạ được chạy trên Google Colab.

## Các năm học

| Năm học | Học kì | Thư mục |
|---------|--------|---------|
| 2025–2026 | 1 | [`2526-1/`](2526-1/) |

> Mỗi năm học mới sẽ được thêm vào dưới dạng một thư mục riêng (ví dụ: `2627-1/`).

## Cấu trúc mỗi năm học

> Cấu trúc dưới đây chỉ mang tính khái quát; một số thư mục có thể có thêm file cấu hình, tài nguyên hoặc script phụ trợ.

```
XXXX-X/
├── index.html                  # Trang chủ (lịch giảng dạy)
├── lecture-*.html              # Slide bài giảng (Reveal.js)
├── lecture-style.css           # CSS dùng chung cho các slide
├── projects/
│   ├── index.html              # Trang tổng hợp các đồ án
│   └── project-*.html          # Mô tả từng đồ án
└── img/
    └── lecXX/                  # Hình minh hoạ cho từng bài giảng
```

## Chạy preview server

Xem hướng dẫn đầy đủ (bao gồm cách chạy bằng Node.js hoặc Python) trong `README.md` của thư mục năm học tương ứng.

## Thêm năm học mới

1. Tạo thư mục mới (ví dụ: `2627-1/`) bằng cách copy từ năm học gần nhất.
2. Cập nhật `index.html` ở root để thêm link đến năm học mới.
3. Xem hướng dẫn chi tiết trong `README.md` của thư mục năm học tương ứng.
